	2. wrapper class
		1. autoboxing 
			- 기본 데이터값이 참조 타입으로 변환되어 대입되는 메카니즘
			Integer i = new Integer(3);
			Integer i2 = 3;		-> 컴파일 -> Integer i2 = new Integer(3);
		
		2. unboxing
			- 객체 타입의 데이터를 기본 타입으로 변환해서 대입 
			int i = new Integer(1); 컴파일 후 int i = (new Integer(1)).intValue();
